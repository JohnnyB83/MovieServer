# Movie Server Project

## Features
  * User accounts and authentication
  * Local comments and movie ratings
  * IMDB info lookup
  * Storage of movie data and thumnail image on MongoDB
  * Simple streaming from local path
  
  ## Dependencies
  * express
  * express-session
  * body-parser
  * ejs
  * method-override
  * mongoose
  * passport
  * passport-local
  * passport-local-mongoose
  * imdb-api
